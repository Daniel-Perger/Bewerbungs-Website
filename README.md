# Bewerbungs-Website
## Beschreibung
Diese Website habe ich, bzw. werde ich grundsätzlich für den Sinn und Zweck Programmieren, um mich für den Beruf des Applikationsentwickler zu bewerben, spezifischer für eine Lehrstelle.
Des Weiteren ist dies für mich ein perfektes kleines Projekt, um meine Leidenschaft für das Programmieren zu vertiefen.

## Inhalte
Die Website beinhaltet folgendes:
- Lebenslauf
- Kontakt

## Benutzte Programmiersprachen:
- HTML
- JavaScript
- CSS

## Ziel
Ich möchte mich mit dieser Website nicht einfach rein bewerben und sehe es daher nicht als "Bewerbungsprozess", ich sehe es eher als ein Projekt ein wunderbares Hobby zu verfolgen. D.h. ich beginne damit, die Grundzüge von HTML, JavaScript und CSS zu erlernen und später zu vertiefen.
